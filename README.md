## 📌 Project Overview
Traditional Air Quality Stations are expensive and only provide city-wide averages. Our solution is a low-cost IoT node network that provides ward-by-ward, real-time pollution data. We utilize a Machine Learning model to analyze PM2.5 and PM10 ratios to instantly identify the **source** of the pollution (e.g., Construction Dust vs. Biomass Burning) and trigger automated mitigation policies.
